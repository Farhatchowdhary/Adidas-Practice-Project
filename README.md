Adidas Practice Project
Project Overview

This project is a practice clone of an Adidas website built with HTML, CSS, and JavaScript. It features a responsive navigation bar, product sections, and interactive elements such as search validation, a shopping cart modal, a login modal, and a heart toggle for favorites. The project is intended for learning and practicing front-end web development skills.

Features

Navigation Bar

Multi-level dropdowns for promotions and seasonal messages.

Links for Men, Women, Kids, Gifts, Sale, and New & Trending sections.

Interactive Icons

Search icon with input validation (alerts if the search term is less than 3 characters).

Heart icon that toggles a favorite state.

Shopping cart icon with a modal showing cart status.

Login Modal

Click the login icon to open a modal with multiple login options (Apple, Google, Facebook, Yahoo).

Email input validation with a “Continue” button that displays an alert.

Product and Image Sections

Large hero image for promotions.

Multiple product sections with images and descriptions.

Popular brands and seasonal items listed with categories.

Footer

Multiple informational sections (Products, Sports, Company Info, Support, Collection, Follow Us).

Privacy footer and copyright notice.

Technologies Used

HTML – for structure and content.

CSS – for styling and layout.

JavaScript – for interactivity and DOM manipulation.

How to Run

Clone or download the repository.

Open the index.html file in your web browser.

Interact with the website:

Click on the heart icon to toggle favorite.

Use the search bar and click the search icon to see validation alerts.

Click the cart icon to view the cart modal.

Click the login icon to open the login modal, enter an email, and click Continue for validation alerts.
